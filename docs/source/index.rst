causalchamber documentation
===========================

The ``causalchamber`` package provides different functionality for the `Causal Chambers <https://causalchamber.ai>`_:

- **Remote Lab**: a Python interface to remotely access our pool of chambers and run your own experiments.
- **Datasets**: download existing datasets from the `dataset repository <https://github.com/juangamella/causal-chamber>`_ directly into your Python code.
